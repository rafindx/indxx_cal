Manual Processing

Operational Risks

Missing a corporate action or a date, misinterpreting/mismanaging an event, using incorrect or conflicting data, or miscalculating an entitlement are just a few of the operational, reputational, and regulatory risks that come with the manual processing of corporate actions. Many of these issues can be addressed by combining data with modern technology, automating procedures, and implementing best practices.

CAdmin mitigates the costly risk of missing discrepancies in corporate actions data by collating, standardizing, validating, and finally consolidating them into a single consistent data copy.

Security Universe

CAdmin enables you to build a universe of securities for which you'd like to track daily corporate actions. This is a one-time task that must be completed when launching CAdmin and adding/removing securities from the universe.

Compliance & Security

All the data that is expected to be handled in CAdmin on a daily basis will be provided by the client and will only be stored on the client's servers. Clients would have access to their data only.

Manual CA insertion

If your data providers fail to report a company's corporate action on a given day, CAdmin allows you to manually enter that corporate action.

User-specific Dashboard

CAdmin allows you to build a dashboard based on your needs and resources, and plan accordingly to take your required critical decisions.

Analytics

CAdmin provides a graphical summary dashboard that provides performance of key indicators at-a-glance, helpful in planning, decision making, and managing activities.

FREQUENTLY ASKED QUESTIONS?

  • Who provides the access to data?

    Clients provide the data that is supposed to be handled in CAdmin using their existing data fees. This allows minimal changes to their back-end systems and processes. CAdmin will only provide a framework for efficiently managing all of that data.

  • How does CAdmin ensures data privacy?

    All of the data that is expected to be handled in CAdmin on a daily basis will stay on that specific client’s servers. CAdmin will have access to client’s data on their servers, and all the processed data will be stored solely on the client's servers.

  • What is the maximum number of data aggregators that CAdmin can handle?

    CAdmin can work with any number of data aggregators. Most client utilize CAdmin for 2-5 data aggregators.

  • Is it possible to track the corporate action of any publicly traded corporation in the world?
